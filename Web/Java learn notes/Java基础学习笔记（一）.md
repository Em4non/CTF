# Java程序设计环境

 - JDK：编写Java程序的程序员使用的软件。
 - SE：用于桌面或简单服务器应用的Java平台。
 - OpenJDK：Java SE的一个免费开源实现。

编译运行Java程序：

```powershell
javac xxx.java
java xxx
```
